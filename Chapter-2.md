    |    False     |     False    |     False    |

                       ### or    
    | Expression-1 | Expression-2 | Evaluates to |
    | ------------ | ------------ | ------------ |
    |    True      |     True     |     True     |
    |    True      |     False    |     True     |
    |    False     |     True     |     True     |
    |    False     |     False    |     False    |
                      
                       ### not                        
    | Expression-1 | Evaluates to |
    | ------------ | ------------ |
    |    True      |     False    | 
    |    False     |     True     |
